## 5. On-page SEO checklist (per page)

Use this for every important URL before launch:

- [ ] Unique `<title>` (50–60 chars, keyword near front)
- [ ] Meta description (150–160 chars, includes CTA)
- [ ] One clear H1 matching search intent
- [ ] Logical H2/H3 hierarchy
- [ ] Internal links to related pages
- [ ] Optimised images (alt text, compressed, modern formats)
- [ ] Canonical URL set
- [ ] Open Graph + Twitter cards

---

## 6. Technical SEO

- [ ] Mobile-first responsive design

- [ ] HTTPS only
- [ ] Clean URL slugs (no IDs in public URLs where avoidable)
