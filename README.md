1. Clone this repo (or open the folder in Cursor).
2. Review and contribute to the SEO plan in [`docs/SEO_PLAN.md`](docs/SEO_PLAN.md).
3. Implementation details will be added once the plan is agreed.

```bash
cd the-bright-brand
```
